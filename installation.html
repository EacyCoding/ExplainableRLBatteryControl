InstallationÔÉÅ

Install the latest CityLearn version from PyPi with the pip command:

pip install CityLearn

Alternatively, install a specific version:

pip install CityLearn==1.4.4

Warning

Version 1.1.0 and earlier are not installable via pip. Instead, clone earlier versions like:

git clone -b v1.1.0 https://github.com/intelligent-environments-lab/CityLearn.git
